函数

Harvey Xie 2023-12-10 11:43:00
Categories： C Tags： 语言基础

C 语言是一门典型的面向过程的编程语言。在面向过程中，会在代码代码中大量的使用到函数。函数也可以称为方法、程式等。在 C 语言中，每个 C 程序有且只有一个 main 主函数，所有的代码都从主函数开始执行。函数是程序的基本模块，用于将特定的功能聚合到一块，以便提高程序的模块化和可复用性。

系统函数和自定义函数

从函数的定义角度划分，函数可以分成系统函数和自定义函数两种：

  • 系统函数

    也称为库函数。由编译器系统提供的函数，用户先导入头文件后，可以直接使用它们，不需要额外的定义。这类函数的头文件通常使用尖括号 < >来 导入，例如：#include <stdio.h>。

  • 自定义函数

    用户为了实现某些功能，自己定义和封装的函数。这类函数的头文件通常使用双引号 " " 来导入，例如#include "mytools.h"，其中 mytools 是用户自定义的一个头文件。

定义

在 C 语言中，定义一个函数的一般形式如下：

return_type function_name( parameter list )
{
   body of the function
}

函数由一个函数头和一个函数主体组成。下面列出一个函数的所有组成部分：

  • 返回类型：一个函数可以返回一个值。return_type 是函数返回的值的数据类型。有些函数执行所需的操作而不返回值，在这种情况下，return_type 是关键字 void。函数返回类型决定返回值的类型，对于数值数据，编译器会隐式转换。若隐式转换失败，则报异常。
  • 函数名称：这是函数的实际名称。函数名和参数列表一起构成了函数签名。
  • 形参列表：形参列表就像是占位符。当函数被调用时，您向参数传递一个值，这个值被称为实际参数。参数列表包括函数参数的类型、顺序、数量。参数是可选的，也就是说，函数可能不包含参数。也就是说形参列表为空的话，可以什么都不写，也可以写一个 void，例如void max(void)。
  • 函数主体：函数主体包含一组定义函数执行任务的语句。

声明

与其他编程语言有区别的是，C 语言中，如果使用自定义的函数，并且该函数与主函数不在同一个文件中，或者函数定义的位置在主函数以后，那么必须在调用此函数之前对被调用的函数进行声明。所谓函数声明，就是在函数尚在未定义的情况下，事先将该函数的有关信息通知编译系统，相当于告诉编译器，函数在后面定义，以便使编译能正常进行。

#include <stdio.h>

//这里就是函数的声明
int sub(int a, int b);

int add(int a, int b)
{
    return a + b;
}

int main()
{
    printf("sum:%d\n", add(5, 3));
    printf("value:%d\n", sub(5, 3));
}

int sub(int a, int b)
{
    return a - b;
}

一个函数只能被定义一次，但是可以声明多次。

函数声明与函数定义的区别：

  1. 函数的定义是对函数的功能的确定，确定的过程包括函数名、函数类型、形参列表、函数体等。函数的定义是一个完整的、独立的函数单位；
  2. 函数的声明的作用是把函数的名字、函数的类型以及形参列表等（不包括函数体）告知编译系统，以便在对包含的函数编译的时候，进行对照检查。

调用

函数定义完成后，可以在程序适当的位置进行函数的调用。

#include <stdio.h>

int add(int a, int b)
{
    return a + b;
}

int main()
{
    printf("sum:%d\n", add(5, 3));
}

头文件

当函数很多时候，需要把一个文件拆分成多个文件，那么此时就需要引用其他函数的头文件。例如：

头文件 methods.h：

#include "methods.c"

int max(int a, int b);

函数实现文件 methods.c：

int max(int x, int y)
{
    return x > y ? x : y;
}

主函数文件 Max.c：

#include <stdio.h>
#include "methods.h"

int main()
{
    printf("%d", max(5, 10));
}

当一个项目比较大时，往往都是分文件，这时候有可能不小心把同一个头文件 include 多次，或者头文件嵌套包含。此时编译的时候，就会报错。因此，为了避免同一个文件被 include 多次，有两种方式解决该问题：第一种是#ifndef 方式，另外一种是 #pragma once 方式。

  1. #ifndef

    #ifndef YOUR_HEADER_FILE_H
#define YOUR_HEADER_FILE_H

// 声明语句

#endif

    这种方式利用了条件编译预处理指令，当 YOUR_HEADER_FILE_H 未定义时，执行 #define 定义该标识符，并包含头文件内容；当 YOUR_HEADER_FILE_H 已经定义时，跳过头文件内容。这样可以避免同一个头文件被多次包含。

  2. #pragma once

    #pragma once

// 声明语句

    使用 #pragma once 可以简化防止头文件被多次包含的操作，但是需要注意的是，它并非标准的 C/C++ 语法，因此在跨平台开发时可能会存在兼容性问题。

总的来说，两种方式都可以有效地避免同一个头文件被多次包含。#ifndef 方式是标准的 C/C++ 语法，在所有编译器中都能够使用；而 #pragma once 方式在一些编译器中也被广泛支持，但不属于官方的 C/C++ 标准。

全局函数和静态函数

根据函数的类型，可以成为：全局函数和静态函数。

在 C 语言中，函数默认都是全局的，只有使用了关键字 static 修饰的函数才是静态函数。静态函数的作用域在该函数所在的文件中有效，无法在其他文件中调用，即使在其他文件中声明这个函数也无法使用。

所有的函数默认都是全局的，意味着所有的函数都不能重名，但如果是静态函数，那么作用域是文件级的，所以不同的文件 static 函数名是可以相同的。

输入输出函数

C语言中常见的输入输出函数有：putchar、getchar、printf 和 scanf。

putchar

putchar 函数是字符输出函数，其作用是在终端控制台输出单个字符。

#include<stdio.h>

int main()
{
    putchar('A');
    putchar('B');
    putchar('\n');
}

getchar

getchar函数的功能是接收用户从键盘上输入的一个字符。

#include <stdio.h>

int main()
{
    char input = getchar();
    printf("%c\n", input);
}

printf

printf函数也叫做格式输出函数，其作用是按照指定的格式输出数据到终端控制台。

格式符 解释说明
%c 输出单个字符
%s 输出字符串。从字符串的第一个字符开始，直到\0结束
%d 以十进制形式输出带符号整数(正数不输出符号)
%f 以小数形式输出单、双精度实数
%u 以十进制形式输出无符号整数
%e或%E 以指数形式输出单、双精度实数
%x或%X 以十六进制形式输出无符号整数(不输出前缀Ox)
%o 以八进制形式输出无符号整数(不输出前缀0)
%% 显示一个%。 转义字符’' 对 % 转义无效
%m.n 格式化输出浮点型。一共有 m 位(整数、小数、小数点)，n位小数
%0m.nf 格式化输出浮点型。一共有 m 位(整数、小数、小数点)，n位小数。 0：表示不足 m 位时，用0凑够m位
%Ns 显示N个字符的字符串。不足N用空格向左填充。
%0Ns 显示N个字符的字符串。不足N用0向左填充。
%-Ns 显示N个字符的字符串。不足N用空格向右填充。

scanf

scanf函数格式化输入函数，其作用是按照指定的格式，从键盘上读取数据，输入到对应的变量中。使用这个函数时，需要注意几点：

  1. 可以指定数据的宽度，但是不能指定数据的精度；
  2. long类型数据使用%ld，double类型的数据使用%lf或%le。

其输入的格式与printf类似。scanf函数还可以与正则表达式一起使用。

#include <stdio.h>

int main()
{
    char a;
    scanf("%c", &a);

    double b;
    scanf("%lf", &b);

    printf("a = %c\n", a);
    printf("b = %f\n", b);
    
    char str[5];
    scanf("%[^\n]",str);
}

使用scanf函数获取字符串的时候，需要保证字符串的空间足够大，避免溢出。

gets

gets函数从键盘获取一个字符串，返回字符串的首地址，该函数可以获取带有空格的字符串。不推荐使用gets函数，因为该函数存在越界的风险。

#include <stdio.h>

int main()
{
    char str[5];
    gets(str);
    printf("Input:%s", str);
}

fgets

fgets函数从标准输入stdin中获取一个字符串，预留\0的存储空间，当空间足够的时候，读取输入最后的换行符\n，空间不足的时候舍弃\n。

#include <stdio.h>

int main()
{
    char str[5];
    fgets(str, 5, stdin);
    printf("Input:%s", str);
}

puts

puts函数将一个字符串写到屏幕，输出的字符串会自动添加换行符\n。

#include <stdio.h>

int main()
{
   char str[6] = "hello";
   puts(str);
}

fputs

fputs函数将一个字符串写到标准出stdout中，输出的字符串不会自动添加换行符\n。

#include <stdio.h>

int main()
{
    char str[6] = "hello";
    fputs(str, stdout);
}
