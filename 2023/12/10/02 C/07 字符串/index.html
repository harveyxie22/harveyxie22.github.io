字符串

Harvey Xie 2023-12-10 11:43:00
Categories： C Tags： 语言基础

在 C 语言中，字符串是由字符组成的数组。可以使用字符数组来表示字符串，每个字符占据一个数组元素的位置，并以空字符('\0')作为字符串的结束符。

声明

常见的字符串声明和初始化方式有三种：

  1. 声明并初始化字符串数组；

    char str[] = "Hello, World!";

    在声明字符数组时，可以直接使用一对双引号来初始化字符串。这样会自动添加尾部的空字符('\0')作为字符串的结束符。

  2. 声明字符数组后再进行赋值；

    char str[20];
str[0] = 'H';
str[1] = 'e';
str[2] = 'l';
str[3] = 'l';
str[4] = 'o';
str[5] = '\0';  // 字符串的结束符

    在这种方式中，首先需要声明一个长度足够的字符数组，然后逐个赋值，最后手动添加字符串的结束符('\0')。

  3. 使用 strcpy() 函数来复制字符串。

    char str1[] = "Hello";
char str2[10];

strcpy(str2, str1);  // 将 str1 复制到 str2 中

    在这种方式中，使用 strcpy() 函数将一个字符串复制到另一个字符数组中。需要注意的是，目标字符数组必须足够大，以容纳源字符串的内容和结束符('\0')。

字符串与字符数组

字符数组和字符串是 C 语言中的两个概念，它们有一些相同点，但也有一些重要的区别。

  1. 存储方式不同：字符数组是以字符序列的形式存储在内存中的，每个字符占用一个字节，可以通过数组下标进行访问和修改。而字符串则是以字符序列加上结尾的空字符('\0')的形式存储在内存中，也就是说字符串是以字符数组的形式存在的，但最后一个元素是空字符。因此，字符串必须以空字符结尾，否则将无法正确处理。
  2. 字符串具有特殊含义：在 C 语言中，字符串是一种特殊的字符数组，因为字符串必须以空字符结尾。这意味着字符串可以使用 C 语言提供的一系列字符串处理函数，例如 strcpy()、strlen()、strcat() 和 strcmp() 等，这些函数可以方便地对字符串进行操作和处理。而字符数组则没有这些特殊的含义，需要自己编写相关的处理函数。
  3. 表示方式不同：字符数组可以直接使用数组名称来表示整个数组，例如 char array[10]; 表示一个包含10个字符的字符数组。而字符串通常使用一个指向其第一个字符的指针来表示，例如 char *str = "Hello"; 表示一个字符串 “Hello”。
  4. 初始化方式不同：在 C 语言中，字符数组可以通过初始化来赋初值，例如 char array[] = {'a', 'b', 'c'};。而字符串则通常使用一对双引号来进行初始化，例如 char *str = "Hello";。

在实际编程过程中，需要根据具体情况选择合适的数据类型来存储和处理数据。

字符串常量

在 C 语言中，字符串常量是以双引号括起来的字符序列。字符串常量是一种特殊的常量，表示一个固定的字符序列。例如，"Hello" 是一个字符串常量，它由字符 ‘H’、‘e’、‘l’、‘l’ 和 ‘o’ 组成，以空字符('\0')作为结束符。可以使用字符串常量来初始化字符数组或者指针变量，例如：

char str[] = "Hello";  // 使用字符串常量初始化字符数组
char *ptr = "Hello";   // 使用字符串常量初始化指针变量
ptr[1] = 'H'; //ERROR! 只读，不可修改！

需要注意的是，字符串常量在内存中是以只读方式存储的，不能直接修改其中的字符。因此，尽管可以将字符串常量赋值给字符数组，但不推荐直接修改该字符数组的内容。如果需要修改字符串，应该使用字符数组来存储和操作。此外，字符串常量还可以作为函数的参数传递，例如：

printf("Hello, World!");  // 字符串常量作为 printf 函数的参数

在上述示例中，字符串常量 “Hello, World!” 作为 printf 函数的参数进行输出。当使用字符串作为函数参数的时候，由于每个字符自身结尾存在\0，因此不需要像数组那样单独再封装一个长度。

strlen

获取字符串的长度。

#include <stdio.h>
#include <string.h> //字符串库函数

int main() {
    char str[] = "Hello World!";
    printf("%llu",strlen(str));
}

strchr、strrchr、strstr

strchr() 和 strrchr() 用于查找字符，前者返回第一次出现的位置，后者返回最后一次出现的位置。它们的查找方向分别是从左往右和从右往左。strstr() 用于查找子串，返回第一次出现的位置，查找方向是从左往右。

这些函数都是以字符数组作为参数，且返回的是指向字符串中某个位置的指针。如果没有找到目标字符或子串，则返回 NULL。此外，这些函数都是区分大小写的，如果需要不区分大小写的查找，可以使用相应的函数变种，如 strcasestr() 等。

strchr strrchr strstr
函数原型 char *strchr(const char *s, int c) char *strrchr(const char *s, int c) char *strstr(const char *haystack, const char *needle)
功能 在字符串 s 中查找字符 c，并返回第一次出现该字符的地址。 ：在字符串 s 中查找字符 c，并返回最后一次出现该字符的地址。 在字符串 haystack 中查找子串 needle，并返回第一次出现该子串的地址。
方向 从左往右查找。 从右往左查找。 从左往右查找。
返回值 如果找到了字符 c，则返回该字符在字符串中的地址；否则返回 NULL。 如果找到了字符 c，则返回该字符在字符串中的地址；否则返回 NULL。 如果找到了子串 needle，则返回该子串在字符串中的地址；否则返回 NULL。 
#include <stdio.h>
#include <string.h>

int main() {
    char str[] = "Hello World!";
    char *r_ptr = strrchr(str, 'W');
    if (r_ptr != NULL) {
        printf("Found character '%c' at position: %lld\n", *r_ptr, r_ptr - str);
    } else {
        printf("Character not found.\n");
    }

    char *ptr = strchr(str, 'o');
    printf("Found character '%c' at position: %lld\n", *ptr, ptr - str);

    char *str_str = strstr(str, "el");
    printf("Found character '%c' at position: %lld\n", *str_str, str_str - str);

    return 0;
}

strtok

strtok() 是 C 语言中用于分割字符串的函数，它可以将一个字符串按照指定的分隔符拆分成多个子字符串。函数原型如下：

char *strtok(char *str, const char *delim);

str 是要分割的字符串，delim 是分隔符。该函数会将 str 按照 delim 进行分割，并返回第一个子字符串的指针。之后可以通过再次调用 strtok(NULL, delim) 来继续获取下一个子字符串。当所有子字符串都被提取完毕时，函数返回 NULL。

#include <stdio.h>
#include <string.h>

int main() {
    char str[] = "Hello,World,C,C++";
    char *ptr = strtok(str, ",");
    while (ptr != NULL) {
        printf("%s\n", ptr);
        ptr = strtok(NULL, ",");
    }
    return 0;
}

在多次调用 strtok() 函数时，第一次传入待分割的字符串 str，之后传入 NULL 来继续分割。这是因为 strtok() 函数会在内部维护一个静态指针，用于记录当前分割的位置。传入 NULL 表示要继续使用上一次调用的字符串。

另外，需要注意的是 strtok() 函数会修改输入字符串 str，将分隔符替换为 NULL 字符 (‘\0’)。如果不想修改原始字符串，可以先创建一个副本进行操作。

strcpy、strncpy

strcpy() 和 strncpy() 都是 C 语言中用于字符串操作的函数，它们用于将一个字符串复制到另一个字符串中。它们的主要区别在于复制的方式和安全性：

strcpy strncpy
函数原型 char *strcpy(char *dest, const char *src) char *strncpy(char *dest, const char *src, size_t n)
功能 将源字符串 src 的内容复制到目标字符串 dest 中，直到遇到源字符串的结束符 \0 为止，然后返回指向目标字符串 dest 的指针 将源字符串 src 的前 n 个字符复制到目标字符串 dest 中，不会自动添加结束符 \0。如果 src 的长度小于 n，则目标字符串之后的部分会用空字符 \0 进行填充，然后返回指向目标字符串 dest 的指针
注意 如果源字符串的长度超过了目标字符串，可能会导致缓冲区溢出，因此使用时需要保证目标字符串有足够的空间来容纳源字符串。 由于 strncpy() 不会自动添加结束符 \0，因此需要手动在目标字符串的末尾添加结束符以确保字符串的完整性。

strcpy() 和 strncpy() 都用于字符串复制，但需要根据具体情况选择合适的函数。如果目标字符串大小足够大并且想要复制整个源字符串，可以使用 strcpy()；如果只想复制源字符串的一部分，并希望限制复制的长度，可以使用 strncpy()。在使用这两个函数时，需要特别注意目标字符串的长度和内存安全性，避免发生缓冲区溢出等问题。

#include <stdio.h>
#include <string.h>

int main() {
    char str[] = "Hello,World,C,C++";
    char dest[20];
    strcpy(dest, str);
    printf("%s\n",str);
    printf("%s\n",dest);

    char dest2[100];
    strncpy(dest2,str,20);
    printf("%s\n",dest2);
}

strcat、strncat

strcat() 和 strncat() 是 C 语言中用于字符串拼接操作的函数，它们用于将一个字符串追加到另一个字符串的末尾。

strcat strncat
函数原型 char *strcat(char *dest, const char *src) char *strncat(char *dest, const char *src, size_t n)
功能 将源字符串 src 追加到目标字符串 dest 的末尾，然后返回指向目标字符串 dest 的指针。 将源字符串 src 的前 n 个字符追加到目标字符串 dest 的末尾，然后返回指向目标字符串 dest 的指针。
注意 使用 strcat() 前需要确保目标字符串 dest 具有足够的空间来容纳源字符串 src。此外，目标字符串 dest 必须以空字符 \0 结尾。 和 strcat() 类似，使用 strncat() 前需要确保目标字符串 dest 具有足够的空间来容纳源字符串 src，并且目标字符串 dest 必须以空字符 \0 结尾。此外，如果源字符串 src 的长度小于 n，则 strncat() 会复制整个源字符串；如果源字符串的长度大于等于 n，则只会复制前 n 个字符。

strcpy() 和 strncpy() 都用于字符串复制，但需要根据具体情况选择合适的函数。如果目标字符串大小足够大并且想要复制整个源字符串，可以使用 strcpy()；如果只想复制源字符串的一部分，并希望限制复制的长度，可以使用 strncpy()。在使用这两个函数时，需要特别注意目标字符串的长度和内存安全性，避免发生缓冲区溢出等问题。

#include <stdio.h>
#include <string.h>

int main() {
    char dest[20] = "Hello";
    const char *src = " World";

    printf("Before strcat:%s\n", dest);
    strcat(dest, src);
    printf("After strcat:%s\n", dest);

    char dest2[20] = "Hello";
    const char *src2 = " World";
    printf("Before strcat:%s\n", dest2);
    strncat(dest2, src2, 2);
    printf("After strcat:%s\n", dest2);
}

strcmp、strncmp

strcmp() 和 strncmp() 是 C 语言中用于字符串比较的函数，它们用于比较两个字符串的内容是否相等。

strcmp strncmp
函数原型 int strcmp(const char *str1, const char *str2) int strncmp(const char *str1, const char *str2, size_t n)
功能 比较字符串 str1 和 str2 的内容。如果两个字符串相等，则返回值为 0；如果 str1 小于 str2，则返回一个负数；如果 str1 大于 str2，则返回一个正数。 比较字符串 str1 和 str2 的前 n 个字符的内容。如果两个字符串的前 n 个字符都相等，则返回值为 0；如果 str1 小于 str2，则返回一个负数；如果 str1 大于 str2，则返回一个正数。
注意 strcmp() 函数会逐个字符比较两个字符串，并根据 ASCII 值来决定大小关系。 和 strcmp() 类似，strncmp() 函数也会逐个字符比较两个字符串，并根据 ASCII 值来决定大小关系。不同之处在于，strncmp() 只比较指定数量的字符，而不是整个字符串。 
#include <stdio.h>
#include <string.h>

int main() {
    const char *str1 = "Hello";
    const char *str2 = "World";

    int result1 = strcmp(str1, str2);
    printf("%d\n", result1);

    const char *str3 = "Hello";
    const char *str4 = "Hello, World!";
    int result2 = strncmp(str3, str4, 5);
    printf("%d\n", result2);
}

atoi、atof、atol

atoi()、atof() 和 atol() 是 C 语言中用于字符串转换为数值类型的函数，它们将一个字符串作为参数，并将其转换为对应的整型、浮点型或长整型数据。

atoi atof atol
函数原型 int atoi(const char *str) double atof(const char *str) long atol(const char *str)
功能 将字符串 str 转换为整型数据。 将字符串 str 转换为双精度浮点型数据。 将字符串 str 转换为长整型数据。
注意 atoi() 函数会忽略字符串开头的空白字符，并且只会将有效数字部分转换成整型。如果字符串中存在非数字字符，则后面的字符将被忽略。 atof() 函数会忽略字符串开头的空白字符，并且只会将有效数字部分转换成双精度浮点型。如果字符串中存在非数字字符，则后面的字符将被忽略。 atol() 函数会忽略字符串开头的空白字符，并且只会将有效数字部分转换成长整型。如果字符串中存在非数字字符，则后面的字符将被忽略。 
#include <stdio.h>
#include <stdlib.h>

int main() {
    const char *str1 = "123";
    int num1 = atoi(str1);
    printf("atoi: %d\n", num1);
    
    const char *str2 = "3.14";
    double num2 = atof(str2);
    printf("atof: %f\n", num2);
    
    const char *str3 = "2147483648";
    long num3 = atol(str3);
    printf("atol: %ld\n", num3);
    
    return 0;
}

需要注意的是，如果字符串中的数字超出了数据类型的范围，则结果可能会不准确。并且当字符串无法转换为对应的数据类型时，这些函数的返回值会是 0 或者未定义的结果，因此在使用时要格外小心。

memcpy、memset、memcmp

atoi()、atof() 和 atol() 是 C 语言中用于字符串转换为数值类型的函数，它们将一个字符串作为参数，并将其转换为对应的整型、浮点型或长整型数据。

memcpy memset memcmp
函数原型 void *memcpy(void *dest, const void *src, size_t n) void *memset(void *s, int c, size_t n) int memcmp(const void *s1, const void *s2, size_t n)
功能 将源内存区域的前 n 个字节复制到目标内存区域。 将指定内存块的前 n 个字节设置为特定的值。 比较两个内存区域的前 n 个字节。
注意 memcpy() 函数用于在内存之间进行字节级别的复制。它可以用来将一个内存块的内容复制到另一个内存块，不受数据类型影响，只是简单的按字节复制。 memset() 函数常用于将一块内存初始化为特定的值，通常是用来初始化字符数组或者设置特定的标记位。 memcmp() 函数用于按字节比较两个内存块的内容。它返回一个整数，表示两个内存块的大小关系，如果相等则返回 0，如果第一个内存块小于第二个，则返回负数，反之返回正数。 
#include <stdio.h>
#include <string.h>

int main() {
    char src[] = "Hello, World!";
    char dest[20];

    memcpy(dest, src, strlen(src) + 1);
    printf("%s\n", dest);

    memset(dest, 'A', 5);
    printf("%s\n", dest);

    int result = memcmp(src, dest, 5);
    printf("memcmp:%d\n", result);
}
