文件

Harvey Xie 2023-12-16 15:49:00
Categories： C Tags： 语言基础

文件是操作系统管理数据的基本单位，也是程序对文件读和写的基本对象。计算机上的各种资源由操作系统管理和控制，专门负责管理设备中文件的系统称之为文件系统。文件一般指的是在外部存储介质上一系列相关数据的有序集合。

文件类型

计算机的存储在物理上是通过二进制实现的，因此理论上所有的文件的本质都是以字节位单位进行顺序存储的二进制文件。但是从用户和操作系统使用的角度（也就是从逻辑上），根据文件的数据组织形式可以分成文本文件和二进制文件。

  • 文本文件

    基于字符编码的文件。把需要存储的数据当成一系列的字符组成。或是把每个字符的 ASCII 码值存入到文件中，或是通过 UNICODE 编码的形式存储字符到文件中。若是以 ASCII 码值存储，则每个 ASCII 码值占一个字节（每个字节表示一个字符）。不管怎样，文本文件也称为字符文件或字符序列文件。

  • 二进制文件

    基于值编码的文件。把数据以二进制的形式存储到文件中，属于字节序列文件。

例如，存在一个数据123，把123分别当做文本文件和二进制文件存储就是：

字符 ‘1’ ‘2’ ‘3’
ASCII（十进制） 49 50 51
ASCII（二进制） 0011 0001 00110010 00110011

如果按照二进制文件形式存储，则把数据 123 看成整型数，如果该系统中整型数占 4 个字节，则数据 123 二进制存储形式的 4 个字节如下：

0000 0000 0000 0000 0000 0000 0111 1011

指针

文件指针

C语言中，用一个指针变量指向一个文件，这个指针称为文件指针。

typedef struct
{
	short           level;	//缓冲区"满"或者"空"的程度 
	unsigned        flags;	//文件状态标志 
	char            fd;		//文件描述符
	unsigned char   hold;	//如无缓冲区不读取字符
	short           bsize;	//缓冲区的大小
	unsigned char   *buffer;//数据缓冲区的位置 
	unsigned        ar;	 //指针，当前的指向 
	unsigned        istemp;	//临时文件，指示器
	short           token;	//用于有效性的检查 
}FILE;

FILE 是系统使用 typedef 定义的有关文件信息的一个结构体，结构体中包含文件的基本元数据信息。声明文件指针结构体的头文件是在 #include <stdio.h> 中。在程序开发过程中，一般会声明和初始化一个文件指针，该指针指向系统中的某个文件。通过文件指针的方式，来对文件进行读、写或追加等操作。

C 语言中，有三个特殊的文件指针由系统默认打开，用户无需定义就可以直接使用。

文件指针 含义
stdin 标准输入。默认为当前终端（键盘），scanf、getchar等函数默认从此终端获取数据。
stdout 标准输出。默认为当前终端（屏幕），printf、puts等函数默认输出的信息到此终端。
stderr 标准出错。默认为当前终端（屏幕），perror等函数默认输出信息到此终端。

文件状态指针

想要获取文件的状态，可以借助结构体 stat，该结构的结构如下：

struct stat
{
    dev_t st_dev;             //文件的设备编号
    ino_t st_ino;             //节点
    mode_t st_mode;           //文件的类型和存取的权限
    nlink_t st_nlink;         //连到该文件的硬连接数目，刚建立的文件值为1
    uid_t st_uid;             //用户ID
    gid_t st_gid;             //组ID
    dev_t st_rdev;            //(设备类型)若此文件为设备文件，则为其设备编号
    off_t st_size;            //文件字节数(文件大小)
    unsigned long st_blksize; //块大小(文件系统的I/O 缓冲区大小)
    unsigned long st_blocks;  //块数
    time_t st_atime;          //最后一次访问时间
    time_t st_mtime;          //最后一次修改时间
    time_t st_ctime;          //最后一次改变时间(指属性)
};

以下是一个获取文件的大小的例子：

#include <stdio.h>
#include <sys/stat.h> //导入该头文件

int main(void)
{
    struct stat st;
    stat("HelloWorld.txt", &st);
    // rename("f.txt", "f_new.txt"); 重命名文件
    // remove("f_new.txt"); 删除文件
    printf("%ld", st.st_size);
}

文件结束标志

对于文本文件而言，EOF 表示文件的结束符 (end of file)。因为在文本文件中，数据以字符的 ASCII 码值的形式进行存放，ASCII 码值的范围是 0~127，因此不会出现 -1 的情况，而此时使用让 EOF 等于-1，就可以让其表示文件结束。

#define EOF (-1)

但是当文件是二进制文件的时候，就无法使用 EOF 的模式判断文件的结束，而是使用 feof 函数来判断文件是否结束。当然，在设计 feof 函数的时候，若只针对二进制文件会显得比较险隘，因此该函数同样可以使用在文本文件上。

int feof(FILE * stream);

该函数的返回值若是非 0 值，表示已经到文件结束。0 值表示还没有到文件结尾。

文件操作

打开和关闭

文件在使用之前需要先执行打开操作，打开后的文件需要及时关闭。打开函数的声明：

FILE * fopen(const char * filename, const char * mode);

其中：

  1. 参数filename：需要打开的文件名，可以使用绝对路径或者相对路径。

  2. 参数mode：文件的打开的模式。

模式 含义 注意
r 只读 若文件不存在，报错。
w 只写 若文件存在则清空文件，若文件不存在，新建一个文件。
a 追加只写 若文件存在，则位置指针移到文件末尾，在文件尾部追加写人，故该方式不删除原文件数据；若文件不存在，则打开失败。
r+ 读写 文件必须存在。在只读 r 的基础上加 ‘+’ 表示增加可写的功能。
w+ 读写 新建一个文件，先向该文件中写人数据，然后可从该文件中读取数据。
a+ 读写 在a模式的基础上，增加可读功能。
rb 二进制读 以二进制模式打开文件，功能同模式r。
wb 二进制写 以二进制模式打开文件，功能同模式w。
ab 二进制追加 以二进制模式打开文件，功能同模式a。
rb+ 二进制读写 以二进制模式打开文件，功能同模式r+。
wb+ 二进制读写 以二进制模式打开文件，功能同模式w+。
ab+ 二进制读写 以二进制模式打开文件，功能同模式a+。

说明：

  1. `b`是二进制的意思。该模式只有在 Windows 下有效，在Linux下 `r` 和 `rb` 的作用相同；
  2. Unix 和 Linux 下的所有文本文件的行结尾都是以 `\n` 结尾，而 Windows 所有的文本文件行都是以 `\r\n`结尾；
  3. 在 Windows 平台下，以文本文件的形式打开文件，不需要加 `b`。当读文件的时候，系统会将所有的`\r\n`转成 `\n`。当写文件的时候，系统会将 `\n` 转换成 `\r\n `写入。但是，当以二进制文件打开文件的时候，这条规则不成立；
  4. 在 Unix 和 Linux 平台下，文本与二进制模式没有区别，`\r\n` 作为字符原样输入和输出。

关闭文件使用 fclose 方法，其声明如下：

int fclose(FILE * stream);

其中参数为一个文件指针，返回值若成功则返回 0，失败返回 -1。

#include <stdio.h>

int main(void)
{
    FILE *file = NULL;
    file = fopen("../HelloWorld.txt", "w+");

    if (file == NULL)
    {
        perror("Open failed.");
        return -1;
    }

    printf("Open a file successfully.\n");
    fclose(file);
    return 0;
}

按字符读写

读

按照字符的方式读文件，可以使用 fgetc 函数。

#include <stdio.h>

int main(void)
{
    FILE *file = NULL;
    file = fopen("HelloWorld.txt", "w+");

    if (file == NULL)
    {
        perror("Open failed.");
        return -1;
    }

    char ch;
    while ((ch = fgetc(file) != EOF))
    {
        printf("%c", ch);
    }
    printf("\n");

    if (feof(file))
    {
        printf("The file has reached the end.");
    }

    fclose(file);
    return 0;
}

写

按照字符的方式写文件，可以使用 fputc 函数。

#include <stdio.h>

int main(void)
{
    FILE *file = NULL;
    file = fopen("HelloWorld.txt", "w+");

    if (file == NULL)
    {
        perror("Open failed.");
        return -1;
    }

    char ch = 'a';
    for (int i = 0; i < 26; i++)
    {
        fputc(ch, file);
        ch++;
    }

    fclose(file);
    return 0;
}

按行读写

读

按照字符的方式读文件，可以使用 fgets 函数。

#include <stdio.h>

int main(void)
{
  FILE *file = NULL;
  file = fopen("helloc.txt", "r");
  if (file == NULL)
  {
    perror("Open failed.");
    return -1;
  }

  printf("Open a file successfully.\n");

  char buff[100] = {0};
  while (!feof(file))
  {
    char *p = fgets(buff, sizeof(buff), file);
    if (p != NULL)
    {
      printf("buff=%s", buff);
    }
  }

  fclose(file);
  return 0;
}

写

按照字符的方式写文件，可以使用 fputs 函数。

#include <stdio.h>

int main(void)
{
  FILE *file = NULL;
  file = fopen("helloc.txt", "w+");
  if (file == NULL)
  {
    perror("Open failed.");
    return -1;
  }

  printf("Open a file successfully.\n");

  char *string[] = {
      "Hello",
      "World",
      "C",
      "\0"};

  for (int i = 0; i < 4; i++)
  {
    fputs(string[i], file);
  }

  fclose(file);
  return 0;
}

按块读写

读

按照字符的方式读文件，可以使用 fread 函数。

#include <stdio.h>

typedef struct Student
{
    char name[50];
    int id;
};

int main(void)
{
    FILE *file = NULL;
    file = fopen("HelloWorld.txt", "r");

    if (file == NULL)
    {
        perror("Open failed.");
        return -1;
    }

    struct Student students[3];
    
    fread(students, sizeof(struct Student), 3, file);

    for (int i = 0; i < 3; i++)
    {
        printf("My name is %s and my id is %d.\n", students[i].name, students[i].id);
    }

    fclose(file);
    return 0;
}

写

按照字符的方式写文件，可以使用 fwrite 函数。

#include <stdio.h>

typedef struct Student
{
    char name[50];
    int id;
};

int main(void)
{
    FILE *file = NULL;
    file = fopen("students.txt", "w");

    if (file == NULL)
    {
        perror("Open failed");
        return -1;
    }

    struct Student students[3];
    for (int i = 0; i < 3; i++)
    {
        sprintf(students[i].name, "stu#%d,", i);
        students[i].id = i + 1;
    }

    fwrite(students, sizeof(struct Student), 3, file);

    fclose(file);
    return 0;
}

格式化读写

读

按照字符的方式写文件，可以使用 fprintf 函数。

#include <stdio.h>

int main(void)
{
    FILE *file = NULL;
    file = fopen("f.txt", "r");

    if (file == NULL)
    {
        perror("Open failed.");
        return -1;
    }

    int num2;
    float pi2;
    char str2[100];

    fscanf(file, "Number:%d\n", &num2);
    fscanf(file, "Pi:%f\n", &pi2);
    fscanf(file, "String:%s\n", str2);

    printf("Number:%d\n", num2);
    printf("Pi:%.2f\n", pi2);
    printf("String:%s\n", str2);

    fclose(file);

    return 0;
}

写

按照字符的方式写文件，可以使用 fprintf 函数。

#include <stdio.h>

int main(void)
{
    FILE *file = NULL;
    file = fopen("f.txt", "r");

    if (file == NULL)
    {
        perror("Open failed.");
        return -1;
    }

    int num = 10;
    float pi = 3.1415926;
    char str[] = "Hello,World!";

    fprintf(file, "Number:%d\n", num);
    fprintf(file, "Pi:%.2f\n", pi);
    fprintf(file, "String:%s\n", str);
    
    fclose(file);

    return 0;
}

随机读写

文件的随机读写常用的三个函数是：fseek、ftell 和 rewind。

  1. fseek

    int fseek(FILE *stream, long offset, int whence);

    该函数的功能是移动文件流（文件光标）的读写位置，成功返回0，失败返回-1。参数 stream 是一个需要操作的文件指针。参数 offset 是根据 whence 为参考坐标的偏移量。offset 的值为正，则相对于 whence 右移动。offset 的值为负，则相对于 whence 左移动。若向前移动的字节数超过了文件开头则出错，若向后移动超过了文件末尾，再次写入时会增大文件尺寸。

    Whence的取值：

    • SEEK_SET：从文件开头移动offset个字节
    • SEEK_CUR：从当前位置移动offset个字节
    • SEEK_END：从文件末尾移动offset个字节

  2. ftell

    long ftell(FILE *stream);

    该函数的功能是获取文件流（文件光标）的读写位置，成功返回当前（文件光标）的读写位置，失败返回-1。

  3. rewind

    void rewind(FILE *stream);

    该函数的功能是把文件流（文件光标）的读写位置移动到文件开头。

以下是一个随机读写的例子：

#include <stdio.h>

typedef struct Student
{
    char name[50];
    int id;
};

int main(void)
{
    FILE *file = NULL;
    file = fopen("HelloWorld.txt", "r");

    if (file == NULL)
    {
        perror("Open failed.");
        return -1;
    }

    struct Student students[3];

    //...假设已经有3个对象被写入到文件中
    
    //跳过前2个元素
    fseek(file, 2 * sizeof(struct Student), SEEK_SET);

    //直接访问第3个元素
    fread(students, sizeof(struct Student) * 1, 1, file);
    printf("My name is %s and my id is %d.\n", students[0].name, students[0].id);

    //在把光标移动到文件开头
    rewind(file); //等同于fseek(file,0,SEEK_SET);

    fread(students,sizeof(struct Student) * 3, 3,file);
    for (int i = 0; i < 3; i++)
    {
        printf("My name is %s and my id is %d.\n", students[i].name, students[i].id);
    }

    fclose(file);
    return 0;
}
