结构体

Harvey Xie 2023-12-19 10:00:00
Categories： C Tags： 语言基础

C 语言中的结构体（Structure）是一种用户自定义的数据类型，用于将不同类型的数据组合在一起形成一个逻辑上相关的数据单元。结构体可以包含多个不同类型的成员变量，这些成员变量可以是基本数据类型、数组、指针或其他结构体。

定义

结构体的定义使用关键字 struct，后面跟着结构体的名称和一对花括号 {}，在花括号内部定义结构体的成员变量。例如：

struct Person {
    char name[20];
    int age;
    float height;
};

上述代码定义了一个名为 Person 的结构体，它有三个成员变量：name、age 和 height。其中，name 是一个字符数组，age 是一个整数，height 是一个浮点数。

有一些结构体会在定义的时候嵌套其他结构体：

struct Class
{
    int grade;
    int class;
    char group;
};

struct Student
{
    char name[50];
    int age;
    long num;
    double score;
    struct Class class;
};

struct Teacher
{
    char *name; // 定义了一个指针
}

在定义结构的时候需要注意：

  1. 结构体可以嵌套另外一个结构体的任何类型变量；
  2. 结构体不可以嵌套本结构的普通变量，因为本结构体的类型大小无法确定；
  3. 结构体可以嵌套本类型的结构体指针变量，因为指针变量的空间是确定的，要么是32位4字节，要么是64位8字节。

访问属性

上述代码定义了一个名为 Person 的结构体，它有三个成员变量：name、age 和 height。其中，name 是一个字符数组，age 是一个整数，height 是一个浮点数。

结构体定义完成后，我们可以通过结构体名加点操作符 . 来访问结构体的成员变量。例如：

struct Person person1;

strcpy(person1.name, "John");
person1.age = 25;
person1.height = 175.5;

上述代码创建了一个名为 person1 的结构体变量，并给其成员变量赋值。通过点操作符 .，我们可以分别访问并操作结构体的每个成员变量。

也可以先定义一个结构体，然后用一个指针指向这个结构体，通过指针来操作结构体成员变量。

#include <stdio.h>
#include <string.h>

struct Class
{
    int grade;
    int class;
    char group;
};

struct Student
{
    char name[50];
    int age;
    long num;
    double score;
    struct Class class;
};

int main(void)
{
    struct Student stu =
        {
            "Lily",
            20,
            10801,
            79.90,
            6,
            5,
            'c'};

    struct Student *stu_ptr = &stu;
    printf("My name is %s and I am in %c group, Class %d, Grade %d\n",
           stu_ptr->name, stu_ptr->class.group, stu_ptr->class.class, stu_ptr->class.grade);
    
    struct Student *p = NULL;
    p = (struct Student *)malloc(sizeof(struct Student));
    strcpy(p->name, "Lily");
    p->age = 18;
    printf("My name is %s and I am %d years old.\n", p->name, p->age);
}

别名

此外，还可以使用 typedef 关键字简化结构体的定义，并给结构体类型取一个别名。例如：

typedef struct {
    char name[20];
    int age;
    float height;
} Person;

上述代码定义了一个匿名结构体，并使用 typedef 将其类型取名为 Person。这样，我们就可以直接使用 Person 来定义结构体变量，而无需再写 struct 关键字。

函数参数

结构体可以作为函数的参数进行传递。当直接传递结构体时，函数是值传递。当传递的时候结构体的指针变量时，函数是地址传递。结构体中若包含数组，则当传递数组名时为地址传递。这是因为数组名本身就是元素的首地址。

#include <stdio.h>
#include <string.h>

struct Student
{
    char *name;
};

void changeNameByValue(struct Student stu)
{
    stu.name = "Eric";
}

void changeNameByAddr(struct Student *stu)
{
    stu->name = "Eric";
}

int main(void)
{
    struct Student stu = {"Mike"};
    changeNameByValue(stu);
    printf("My name is %s.\n", stu.name); //Output: My name is Mike.

    changeNameByAddr(&stu);
    printf("My name is %s.\n", stu.name);//Output: My name is Eric.
}
