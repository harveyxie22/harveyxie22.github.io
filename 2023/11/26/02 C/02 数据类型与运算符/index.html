数据类型与运算符

Harvey Xie 2023-11-26 15:26:00
Categories： C Tags： 语言基础

C 语言是一种静态类型语言，数据类型定义了变量能够存储的数据种类。C 语言中有以下几种数据类型：

  1. 基本数据类型。例如：整型、浮点型、字符型和布尔型；
  2. 复合数据类型。例如：数组、结构体、指针数据类型、枚举数据类型。

基本数据类型

C 语言提供了 4 种基本数据类型，分别是：

  • 整型（Integer）：
    • int：用于表示整数，通常占用 4 个字节（32位系统）或 8 个字节（64位系统），范围为 -2147483648 到2147483647（32位系统）或 -9223372036854775808 到 9223372036854775807（64位系统）。
    • short：用于表示短整数，通常占用 2 个字节，范围为 -32768 到 32767。
    • long：用于表示长整数，通常占用 4 个字节（32位系统）或 8 个字节（64位系统），范围与 int 相同。
    • long long：用于表示更长的整数，通常占用 8 个字节，范围为 -9223372036854775808 到9223372036854775807。
  • 浮点型（Floating-point）：
    • float：用于表示单精度浮点数，通常占用 4 个字节，可以表示大约 6 到 7 位的有效数字。
    • double：用于表示双精度浮点数，通常占用 8 个字节，可以表示大约 15 位的有效数字。
    • long double：用于表示更高精度的浮点数，通常占用 10 或 16 个字节，具体大小和精度因系统而异。
  • 字符型（Character）：
    • char：用于表示单个字符，通常占用 1 个字节。可以表示 ASCII 字符集中的字符，也可以用于存储小整数值。
  • 布尔型（Boolean）：
    • _Bool：用于表示布尔值，只能取 true（非零）或 false（零）两个值。

每种基本数据类型都有自己的取值范围和存储大小，具体取决于编译器和系统。此外，还可以使用限定符来扩展基本数据类型，例如使用 unsigned 修饰符表示无符号整数类型。

整型

实际上，C 语言并没有严格规定 short、int、long 的长度，只做了宽泛的限制：

  • short 至少占用 2 个字节；
  • int 建议为一个机器字长。32 位环境下机器字长为 4 字节，64 位环境下机器字长为 8 字节；
  • short 的长度不能大于 int，long 的长度不能小于 int。

总结起来，它们的长度（所占用的字节数）关系为：

2 ≤ short ≤ int ≤ long

这就意味着，short 并不一定真的“短”，long 也并不一定真的 “长”，它们有可能和 int 占用相同的字节数。

在 16 位环境下，short 的长度为 2 个字节，int 也为 2 个字节，long 为 4 个字节。16 位环境多用于单片机和低级嵌入式系统，在 PC 和服务器上已经见不到了。对于 32 位的 Windows、Linux 和 Mac OS，short 的长度为 2 个字节，int 为 4 个字节，long 也为 4 个字节。PC 和服务器上的 32 位系统占有率也在慢慢下降，嵌入式系统使用 32 位越来越多。 在 64 位环境下，不同的操作系统会有不同的结果，如下所示：

操作系统 short int long
Win64（64位 Windows） 2 4 8
类 Unix 系统（包括 Unix、Linux、Mac OS、BSD、Solaris 等） 2 4 8

目前我们使用较多的PC系统为 Win XP、Win 7、Win 8、Win 10、Mac OS、Linux，在这些系统中，short 和 int 的长度都是固定的，分别为 2 和 4，大家可以放心使用，只有 long 的长度在 Win64 和类 Unix 系统下会有所不同，使用时要注意移植性。

有时候，我们希望自动获取整数的长度，操作符 sizeof 就派上了用场：

#include <stdio.h>
int main()
{
    short a = 10;
    int b = 100;
   
    int short_length = sizeof a;
    int int_length = sizeof(b);
    int long_length = sizeof(long);
    int char_length = sizeof(char);
   
    printf("short=%d, int=%d, long=%d, char=%d\n", short_length, int_length, long_length, char_length);
   
    return 0;
}

在 32 位环境以及 Win64 环境下的运行结果为：

short=2, int=4, long=4, char=1

在 64 位 Linux 和 Mac OS 下的运行结果为：

short=2, int=4, long=8, char=1

sizeof 用来获取某个数据类型或变量所占用的字节数，如果后面跟的是变量名称，那么可以省略 ()。 如果跟的是数据类型， 就必须带上 ()。 需要注意的是，sizeof 是 C 语言中的操作符，不是函数，所以可以不带 ()。

浮点型

浮点型也称为实型。在 C 语言中，实型被分成了 float 单精度、double 双精度和 long double 长双精度。在实际开发过程中，建议多使用 double 类型。

类型 数的精度 字节数
float 6-7位的有效数字 4
double 15-16位有效的数字 8
long double 占用的内存是 double 的两倍，但表达数据的精度和 double 相同 16

字符型

字符型数据使用单引号括起来的一个字符。在 C 语言中，字符型数据有如下的特点：

  1. 字符型数据只能用单引号括起来，不能用双引号或其它括号；
  2. 字符型数据只能是单个字符，不能是字符串；
  3. 字符可以是字符集中任意字符。但数字被定义为字符型之后就不能参与数值运算。如 ‘5’ 和 5 是不同的。‘5’ 是字符型数据，不能参与运算。

字符变量的类型说明符是 char。字符变量类型定义的格式和书写规则都与整型变量相同。例如：
char a,b。每个字符变量被分配一个字节的内存空间，因此只能存放一个字符。字符值是以 ASCII 码的形式存放在变量的内存单元之中的。

在 C 语言允许对整型变量赋以字符值，也允许对字符变量赋以整型值。在输出时，允许把字符变量按整型量输出，也允许把整型量按字符量输出。整型量为二字节量，字符量为单字节量，当整型量按字符型量处理时，只有低八位字节参与处理。

布尔类型

在 C 语言中，布尔类型（Boolean）用于表示真和假的值。C99 标准引入了 _Bool 类型，它可以存储 true 和 false 两个值，分别用 1 和 0 表示。

为了提高代码可读性，C99 标准还定义了一个名为 bool 的宏，它被定义为 _Bool 类型。为了使用 bool 类型，需要包含 stdbool.h 头文件。

#include <stdbool.h>
// 定义 bool 类型变量
bool flag = true;

bool 类型的变量只能存储 true 或 false，不能存储任何其他值。在将非零值转换为 bool 类型时，会被视为 true，将零值转换为 bool 类型时，会被视为 false。

例如：

bool a = 1;    // a = true
bool b = 0;    // b = false
bool c = -1;   // c = true

bool 类型通常用于条件判断和逻辑运算符，如 if、while、for、&& 和 || 等。

例如：

#include <stdbool.h>
#include <stdio.h>

int main() {
    bool a, b;
    printf("请输入两个布尔值（0 或 1）：");
    scanf("%d %d", &a, &b);
    if (a && b) {
        printf("您输入的两个布尔值都为 true。\n");
    } else {
        printf("您输入的两个布尔值中至少有一个为 false。\n");
    }
    return 0;
}

运算符

在 C 语言中，运算符用于对变量和常量进行操作。C 语言支持多种运算符，包括算术运算符、关系运算符、逻辑运算符、位运算符、赋值运算符、条件运算符和逗号运算符等。

以下是 C 语言中常见的运算符：

  1. 算数运算符。例如：加、减、乘、除、取余；
  2. 关系运算符。例如：小于、大于、小于等于、大于等于、等于和不等于；
  3. 逻辑运算符。例如：与、或、非；
  4. 位运算符。例如：按位与、按位或、按位异或、按位取反、左移、右移；
  5. 赋值运算符。例如：赋值、加等于、减等于，等等；
  6. 三目运算符；
  7. 逗号运算符。

算数运算符

+（加）、-（减）、*（乘）、/（除）、%（取余）

int a = 10, b = 3;
int c = a + b;   // c = 13
int d = a - b;   // d = 7
int e = a * b;   // e = 30
int f = a / b;   // f = 3
int g = a % b;   // g = 1

关系运算符

<（小于）、>（大于）、<=（小于等于）、>=（大于等于）、==（等于）、!=（不等于）

int a = 10, b = 3;
bool c = a < b;    // c = false
bool d = a > b;    // d = true
bool e = a <= b;   // e = false
bool f = a >= b;   // f = true
bool g = a == b;   // g = false
bool h = a != b;   // h = true

逻辑运算符

&&（与）、||（或）、!（非）

int a = 10, b = 3, c = 5;
bool d = a > b && b < c;   // d = true
bool e = a < b || b > c;   // e = false
bool f = !(a == b);        // f = true

位运算符

&（按位与）、|（按位或）、^（按位异或）、~（按位取反）、<<（左移）、>>（右移）

int a = 10, b = 3;
int c = a & b;    // c = 2
int d = a | b;    // d = 11
int e = a ^ b;    // e = 9
int f = ~a;       // f = -11
int g = a << 1;   // g = 20
int h = a >> 1;   // h = 5

赋值运算符

=（赋值）、+=（加等于）、-=（减等于）、*=（乘等于）、/=（除等于）、%=（取余等于）、&=（按位与等于）、|=（按位或等于）、^=（按位异或等于）、<<=（左移等于）、>>=（右移等于）

int a = 10, b = 3;
a += b;    // 等价于 a = a + b，a = 13
a -= b;    // 等价于 a = a - b，a = 10
a *= b;    // 等价于 a = a * b，a = 30
a /= b;    // 等价于 a = a / b，a = 10
a %= b;    // 等价于 a = a % b，a = 1
a &= b;    // 等价于 a = a & b，a = 2
a |= b;    // 等价于 a = a | b，a = 3
a ^= b;    // 等价于 a = a ^ b，a = 1
a <<= 1;   // 等价于 a = a << 1，a = 2
a >>= 1;   // 等价于 a = a >> 1，a = 1

三目运算符

int a = 10, b = 3;
int c = a > b ? a : b;   // c = 10

逗号运算符

int a = 10, b = 3;
int c = (a += b, a *= b, a %= b);   // c = 0，a = 0
